Description Générale

Depuis 1997, NetCeler conçoit et déploie des solutions innovantes de monitoring pour les réseaux électriques, l'industrie et les transports, principalement axée autour de 3 produits :

  • IVPower : Solution de Monitoring pour les réseaux électriques
  • IVTracer : Solution de Monitoring des paramètres critiques environnementaux
  • IVPark : Solution de Télé-Exploitation de parkings

  • Assemblage serveur
